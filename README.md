# Gutter Colour

Gutter Color is a Sublime Text plugin which displays a colored icon for all lines which contain a color.

![GutterColor](screenshot.png)

## Requirements

* ImageMagick

## Configure

* Install ImageMagick
* Set the `convert_path` in `Preferences: GutterColor Settings â€“ User` to the location of the ImageMagick `convert` script.
